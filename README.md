# Sistema Gerenciador de Estoque

Este é um sistema para gerenciamento de estoque desenvolvido em Python utilizando PyQt5 para a interface gráfica e SQLite para o armazenamento de dados.

## Funcionalidades

- Cadastro de produtos com informações como nome, descrição, quantidade em estoque e preço.
- Visualização, edição e exclusão de produtos cadastrados.
- Controle de entrada e saída de produtos no estoque.
- Geração de relatórios de estoque.
- Interface amigável e intuitiva.

## Requisitos

- Python 3.x
- PyQt5
- SQLite
